## 💻 Panduan Instalasi dan Menjalankan Lokal

Jika penguji/dosen ingin menjalankan aplikasi ini secara lokal di perangkat masing-masing, silakan ikuti langkah-langkah berikut:

**1. Persiapan Lingkungan (Requirements)**
Pastikan Python sudah terinstal, lalu instal seluruh *library* yang dibutuhkan dengan menjalankan perintah ini di terminal:
`pip install -r requirements.txt`

**2. Menjalankan Aplikasi (Inference)**
Setelah instalasi selesai, jalankan aplikasi web lokal menggunakan Streamlit dengan perintah:
`python -m streamlit run app.py`

**3. Akses Web Lokal**
Buka browser dan akses alamat `http://localhost:8501` untuk melihat antarmuka aplikasi dan melakukan prediksi.
